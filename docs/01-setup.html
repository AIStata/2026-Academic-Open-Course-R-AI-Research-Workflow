2  R 环境与 VS Code 配置

本章介绍如何搭建 R 语言科研工作环境，包括 R、RStudio 和 VS Code 的配置。

2.1 R 与 RStudio 安装

2.1.1 安装 R

从 CRAN 官网 下载适合你操作系统的 R 版本：

  • Windows: 点击 “Download R for Windows”
  • macOS: 点击 “Download R for macOS”
  • Linux: 选择对应的发行版

2.1.2 安装 RStudio

RStudio 是最流行的 R IDE，推荐安装 RStudio Desktop：

  1. 访问 RStudio 官网
  2. 下载 RStudio Desktop (Free版)
  3. 按默认设置安装

2.1.3 验证安装

安装完成后，打开 RStudio，在控制台输入：

R.version.string

如果显示版本号，说明安装成功。

2.2 VS Code R 扩展配置

2.2.1 安装 VS Code

从 VS Code 官网 下载并安装。

2.2.2 安装 R 扩展

在 VS Code 中安装以下扩展：

  1. R Extension Pack (包含核心扩展)

    • R (by REditorSupport)
    • R Debugger
    • R LSP Client

  2. Quarto (用于本书的 .qmd 文档)

  3. Python (可选，用于混合编程)

2.2.3 配置 R 终端

打开 VS Code 设置 (Ctrl+,)，搜索 “Rterm”，配置 R 路径：

  • Windows:

    {
  "r.rterm.windows": "C:/Program Files/R/R-4.x.x/bin/R.exe",
  "r.rterm.option": ["--no-save", "--no-restore"]
}

  • macOS:

    {
  "r.rterm.mac": "/usr/local/bin/R"
}

2.2.4 终端集成测试

创建一个新的 R 脚本文件 test.R，输入：

# 测试代码
print("Hello, R!")
x <- 1:10
mean(x)

选中代码，按 Ctrl+Enter 运行，观察终端输出。

2.3 包管理基础

2.3.1 安装包

使用 install.packages() 安装 CRAN 上的包：

# 安装单个包
# install.packages("tidyverse")

# 安装多个包
# install.packages(c("haven", "panelView", "did"))

提示：国内用户可以配置镜像源加速下载：

# 配置清华镜像
options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))

2.3.2 加载包

使用 library() 加载已安装的包：

# 加载 tidyverse 包
library(tidyverse)

2.3.3 查看已安装的包

# 查看已安装的包列表
# installed.packages()[, c("Package", "Version")]

# 检查特定包是否安装
# requireNamespace("tidyverse", quietly = TRUE)

2.3.4 更新包

# 更新所有包
# update.packages(ask = FALSE)
